# FAQ FloodRisk AI

## Apa itu FloodRisk AI?
Sistem pendukung keputusan risiko banjir untuk hortikultura di Pekanbaru dan sekitarnya. Menggunakan data cuaca Open-Meteo dan model Random Forest.

## Dari mana data cuaca berasal?
Open-Meteo API — data forecast harian termasuk curah hujan, kelembapan, dan temperatur.

## Seberapa akurat prediksi?
Model Random Forest dilatih dari data historis BMKG. Akurasi tergantung kondisi cuaca aktual. Prediksi bersifat estimasi, bukan kepastian.

## Apakah FRI berubah setiap hari?
Ya. FRI dihitung ulang setiap kali pengguna membuka aplikasi berdasarkan forecast terbaru.

## Apakah sistem ini menggantikan keputusan petani?
Tidak. Sistem hanya memberikan informasi dan rekomendasi. Keputusan budidaya tetap berada pada pengguna.

## Apa yang harus dilakukan saat FRI tinggi?
Tunda penanaman, aktifkan mitigasi drainase, lindungi tanaman yang sudah ada, prioritaskan komoditas tahan genangan.
